Nume proiect:Apelarea liftului
Autor:Steau Roxana-Ioana
Descriere: Aplicatia curenta are 3 fire de executie, unul pentru clientul care vine sa ceara liftul, unul pentru liftul care coboara atunci cand 
clientul a ajuns in punctul dorit si unul care face sincronizare. Clientul pleaca, odata ce aplicatia este pornita(incepe rularea), cand ajunge 
in puntul stabilit se opreste, se va seta liftul care va porni, cand ajunge la client se vor seta sincronizarea si clientul si vor porni impreuna.
Tehnologii dolosite: c# .Net Windows Forms Application
